Guia da Opção A: Calculadora de Idade

Estrutura do Algoritmo

Use este guia para criar seu programa em Portugol. Você pode usar uma ferramenta online como o Portugol Webstudio para testar.

1. Início e Variáveis

Todo programa começa com a palavra `inicio`. Você precisará de três variáveis do tipo `inteiro`: uma para o ano de nascimento, uma para o ano atual e uma para a idade.

2. Entrada de Dados

Use o comando `escreva()` para pedir ao usuário que digite o ano em que nasceu. Em seguida, use o comando `leia()` para guardar esse valor na variável correspondente. Faça o mesmo para o ano atual.

3. Processamento

Esta é a parte lógica. A idade será o resultado da subtração do ano atual pelo ano de nascimento. Atribua o resultado dessa conta à sua variável `idade`.

4. Saída de Dados

Use o comando `escreva()` para mostrar o resultado na tela. Você pode montar uma frase, como: "Você tem ou vai fazer X anos em [ano atual]."

Exemplo de Código


programa {
  funcao inicio() {
    inteiro ano_nascimento
    inteiro ano_atual
    inteiro idade

    escreva("Digite o seu ano de nascimento: ")
    leia(ano_nascimento)

    escreva("Digite o ano atual: ")
    leia(ano_atual)

    idade = ano_atual - ano_nascimento

    escreva("\nSua idade é: ", idade, " anos.")
  }
}
