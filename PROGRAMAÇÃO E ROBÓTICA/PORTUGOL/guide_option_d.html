Guia da Opção D: Gerador de PG

Estrutura do Algoritmo

Use este guia para criar um programa que gera os 10 primeiros termos de uma Progressão Geométrica.

1. Início e Variáveis

Declare variáveis do tipo `inteiro` ou `real` para: o primeiro termo, a razão, e o termo atual da sequência.

2. Entrada de Dados

Peça ao usuário para digitar o primeiro termo e a razão da PG. Guarde os valores nas variáveis.

3. Lógica e Laço de Repetição

A lógica é muito parecida com a da PA, mas em vez de somar, vamos multiplicar. Inicialize o `termo_atual` com o valor do primeiro termo. Use um laço `para` de 1 a 10. A cada volta, exiba o `termo_atual` e depois o atualize, multiplicando-o pela razão.

4. Saída de Dados

Dentro do laço, use `escreva()` para mostrar cada termo da PG, separado por um espaço.

Exemplo de Código


programa {
  funcao inicio() {
    real primeiro_termo
    real razao
    real termo_atual

    escreva("Gerador de PG\n")
    escreva("Digite o primeiro termo: ")
    leia(primeiro_termo)

    escreva("Digite a razão: ")
    leia(razao)

    termo_atual = primeiro_termo

    escreva("\nOs 10 primeiros termos da PG são: \n")
    para (inteiro i = 1; i <= 10; i++) {
      escreva(termo_atual, " ")
      termo_atual = termo_atual * razao
    }
  }
}
