Roteiro de Estudos: Lógica de Programação com Portugol

Seus Primeiros Passos no Mundo do Código

O que é Portugol?

Bem-vindos, futuros programadores! Este roteiro é sua porta de entrada para o universo da programação. Vamos usar o Portugol, uma pseudolinguagem que nos permite escrever algoritmos em português, focando na lógica e no raciocínio computacional sem nos preocuparmos com as regras complexas de uma linguagem de programação real. É a ferramenta perfeita para começar a pensar como um programador.

"A lógica é a base de toda a programação. Se você consegue descrever os passos, você consegue programar."

Etapa 1: Fundamentos do Algoritmo

  • O que é um algoritmo? Dê um exemplo de um algoritmo do seu dia a dia (ex: receita de bolo).
  • Qual a estrutura básica de um programa em Portugol (inicio, declaração de variáveis, fim)?
  • O que são variáveis e para que servem? Quais são os tipos de dados mais comuns (inteiro, real, caractere, logico)?
  • Como funcionam os comandos de entrada (`leia()`) e saída (`escreva()`)?
  • Escreva um algoritmo que peça o nome do usuário e depois exiba uma mensagem de boas-vindas com o nome dele.

Etapa 2: Tomando Decisões e Repetindo Tarefas

  • Como funciona a estrutura condicional `se-entao-senao`? Crie um exemplo que verifique se um aluno foi aprovado (média >= 7.0).
  • O que são operadores relacionais (>, <, >=, <=, ==, !=) e lógicos (E, OU, NAO)?
  • Para que servem as estruturas de repetição? Qual a diferença entre `para`, `enquanto` e `faca-enquanto`?
  • Usando a estrutura `para`, crie um algoritmo que escreva os números de 1 a 10 na tela.
  • Modifique o algoritmo anterior para que ele só escreva os números pares.

Etapa 3: Sequências e Progressões

  • O que é uma sequência numérica?
  • Defina Progressão Aritmética (PA). Qual a fórmula para encontrar um termo qualquer de uma PA?
  • Defina Progressão Geométrica (PG). Qual a fórmula para encontrar um termo qualquer de uma PG?
  • Como um laço de repetição pode ser usado para gerar os termos de uma PA ou PG?

Etapa 4: Pequenos Programas - Mãos à Obra!

Com base em seus estudos, escolha e desenvolva um dos seguintes programas em Portugol:

Opção A: Calculadora de Idade

Peça o ano de nascimento do usuário e calcule a idade dele.

Opção B: Gerador de Tabuada

Peça um número e mostre a tabuada de multiplicação desse número (de 1 a 10).

Opção C: Gerador de PA

Peça o termo inicial e a razão de uma PA e gere os 10 primeiros termos.

Opção D: Gerador de PG

Peça o termo inicial e a razão de uma PG e gere os 10 primeiros termos.
