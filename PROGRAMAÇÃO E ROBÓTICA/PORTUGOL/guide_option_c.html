Guia da Opção C: Gerador de PA

Estrutura do Algoritmo

Use este guia para criar um programa que gera os 10 primeiros termos de uma Progressão Aritmética.

1. Início e Variáveis

Você precisará de variáveis do tipo `inteiro` para: o primeiro termo da PA, a razão, e uma variável para guardar o termo atual da sequência.

2. Entrada de Dados

Peça ao usuário para digitar o primeiro termo da PA e a razão. Guarde esses valores nas variáveis correspondentes.

3. Lógica e Laço de Repetição

Inicialize a variável `termo_atual` com o valor do primeiro termo. Use um laço `para` que se repita 10 vezes. Na primeira volta, apenas exiba o `termo_atual`. Nas voltas seguintes, some a razão ao `termo_atual` antes de exibi-lo.

4. Saída de Dados

Dentro do laço, use o comando `escreva()` para mostrar cada termo da PA. Você pode adicionar uma vírgula ou um espaço para separar os números.

Exemplo de Código


programa {
  funcao inicio() {
    inteiro primeiro_termo
    inteiro razao
    inteiro termo_atual

    escreva("Gerador de PA\n")
    escreva("Digite o primeiro termo: ")
    leia(primeiro_termo)

    escreva("Digite a razão: ")
    leia(razao)

    termo_atual = primeiro_termo

    escreva("\nOs 10 primeiros termos da PA são: \n")
    para (inteiro i = 1; i <= 10; i++) {
      escreva(termo_atual, " ")
      termo_atual = termo_atual + razao
    }
  }
}
