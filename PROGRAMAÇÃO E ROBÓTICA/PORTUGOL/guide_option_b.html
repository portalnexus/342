Guia da Opção B: Gerador de Tabuada

Estrutura do Algoritmo

Use este guia para criar um programa que gera a tabuada de um número escolhido pelo usuário.

1. Início e Variáveis

Declare uma variável do tipo `inteiro` para guardar o número que o usuário vai escolher. Você também precisará de uma variável de controle para o laço de repetição, que geralmente chamamos de `i`.

2. Entrada de Dados

Peça ao usuário para digitar o número da tabuada que ele deseja ver. Guarde esse valor na sua variável.

3. Laço de Repetição `para`

A tabuada vai de 1 a 10. Um laço `para` é perfeito para isso. Configure o laço para que a variável `i` comece em 1, vá até 10, e aumente de 1 em 1 a cada passo.

4. Saída de Dados Dentro do Laço

A cada volta do laço, você deve calcular e exibir uma linha da tabuada. Por exemplo: `escreva(numero, " x ", i, " = ", numero * i)`. Isso irá mostrar a multiplicação do número do usuário pelo valor atual de `i`.

Exemplo de Código


programa {
  funcao inicio() {
    inteiro numero

    escreva("Digite um número para ver a tabuada: ")
    leia(numero)

    escreva("\nTabuada do ", numero, ":\n")

    para (inteiro i = 1; i <= 10; i++) {
      escreva(numero, " x ", i, " = ", numero * i, "\n")
    }
  }
}
