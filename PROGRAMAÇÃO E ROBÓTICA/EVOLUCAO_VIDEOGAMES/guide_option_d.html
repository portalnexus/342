Guia da Opção D: Documentação do Projeto MakeCode

Estrutura para Apresentar seu Jogo

Use este guia para criar uma pequena apresentação ou relatório sobre o jogo que você recriou no MakeCode Arcade.

1. O Jogo Original

Apresente o jogo clássico que você escolheu. Fale um pouco da sua história e de como suas mecânicas funcionavam.

2. Minha Recriação

Coloque o link compartilhável do seu projeto MakeCode Arcade aqui. Explique qual mecânica específica você decidiu recriar.

3. Desafios de Programação

Mostre um trecho do seu código (pode ser um print dos blocos) e explique uma parte que foi desafiadora ou interessante de programar.

4. Aprendizados

O que você aprendeu sobre lógica de programação e game design ao recriar essa mecânica?
