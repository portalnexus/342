Guia da Opção A: Criando uma Linha do Tempo Interativa

Estrutura para sua Página Web

Use este guia para estruturar seu código HTML e CSS. O uso de JavaScript para a interatividade é um desafio extra bem-vindo!

1. Estrutura HTML (o esqueleto)

No seu arquivo `index.html`, crie uma estrutura para a linha do tempo. Você pode usar uma lista `

    ` onde cada `
  • ` é um evento.

    <ul class="timeline"> <li> <div class="timeline-badge">1972</div> <div class="timeline-panel"> <h4>Lançamento do Pong</h4> <p>Descrição do evento...</p> </div> </li> <!-- Outros eventos --> </ul>

    2. Estilização com CSS

    No seu arquivo `style.css`, crie os estilos para a classe `.timeline`. Pesquise por "CSS Timeline examples" para ter inspiração de design.

    3. Conteúdo

    Adicione pelo menos 5 eventos importantes de diferentes décadas na sua linha do tempo, com uma breve descrição e uma imagem para cada um.

    4. (Opcional) Interatividade com JavaScript

    Adicione um efeito que revela o conteúdo do evento apenas quando o usuário clica no ano, por exemplo.
