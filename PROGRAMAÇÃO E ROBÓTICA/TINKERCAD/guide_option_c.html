Guia da Opção C: Organizador de Mesa 3D

Estrutura para seu Projeto

Use este guia para projetar e modelar seu organizador de mesa.

1. Análise do Problema e Ideias

Olhe para a sua mesa. O que está desorganizado? Lápis, canetas, clipes, seu celular? Escolha um ou dois itens para organizar. Desenhe no papel algumas ideias de como seu organizador poderia ser.

2. Medidas e Planejamento

Use uma régua para medir os objetos que você quer organizar. Um lápis precisa de um buraco com qual diâmetro? Qual a espessura do seu celular? Anote essas medidas. Elas serão essenciais para a modelagem.

3. Modelagem no Tinkercad

Comece com uma forma sólida como base (um cubo ou cilindro). Para criar os compartimentos, use outras formas (como cilindros ou cubos) no modo "Orifício". Posicione esses orifícios dentro da sua base, usando as medidas que você anotou. Use a ferramenta de alinhamento para garantir que tudo fique reto.

4. Agrupamento e Refinamento

Selecione todos os objetos (a base e os orifícios) e clique em "Agrupar". Isso irá "cavar" os buracos e criar seu organizador. Adicione detalhes: chanfros para suavizar as bordas, seu nome, etc.

5. Documentação

Tire um print do seu modelo 3D finalizado. Escreva um parágrafo explicando qual problema seu organizador resolve e como você usou as ferramentas de agrupamento e orifício para criá-lo.
