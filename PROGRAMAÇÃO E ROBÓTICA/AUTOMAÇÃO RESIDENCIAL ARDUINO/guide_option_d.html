  • Materiais/Componentes: O que você está usando?
  • Montagem/Configuração: Como tudo está conectado?

5. Explicação do Código/Lógica (Slide 9-10)

Se houver código, explique as partes mais importantes. Se for uma lógica, use um fluxograma ou pseudocódigo. Conecte o código/lógica ao que está acontecendo na demonstração.


// Exemplo de Código Arduino para o Seminário
void setup() {
  pinMode(13, OUTPUT); // Define o pino 13 como saída
}

void loop() {
  digitalWrite(13, HIGH); // Liga o LED
  delay(500);             // Espera 0.5 segundo
  digitalWrite(13, LOW);  // Desliga o LED
  delay(500);             // Espera 0.5 segundo
}
