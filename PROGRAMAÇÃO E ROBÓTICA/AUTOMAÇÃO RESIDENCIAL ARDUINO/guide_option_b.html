4. Componentes Necessários

  • Sensores: Ex: Sensor de movimento PIR, sensor de fumaça, sensor de temperatura e umidade.
  • Atuadores: Ex: Relés para controle de iluminação, válvulas para irrigação.
  • Módulos de Comunicação: Ex: Módulo Wi-Fi ESP8266, Bluetooth.
